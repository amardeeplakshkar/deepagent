## Usage

```ts
import { HunterClient } from '@deepagent/hunter'

const hunter = new HunterClient()
const res0 = await hunter.domainSearch('apple.com')
const res1 = await hunter.emailFinder({
  domain: 'transitivebullsh.it',
  first_name: 'travis',
  last_name: 'fischer'
})
```
