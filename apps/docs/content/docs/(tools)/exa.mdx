- package: `@deepagent/exa`
- exports: `class ExaClient`, `namespace exa`
- env vars: `EXA_API_KEY`

- [exa api docs](https://docs.exa.ai)

## Install

<PackageTabs>
{`
\`\`\`bash npm
npm install @deepagent/exa
\`\`\`

\`\`\`bash yarn
yarn add @deepagent/exa
\`\`\`

\`\`\`bash pnpm
pnpm add @deepagent/exa
\`\`\`
`}
</PackageTabs>

## Usage

```ts
import { ExaClient } from '@deepagent/exa'

const exa = new ExaClient()
const res = await exa.search('latest news')
```
