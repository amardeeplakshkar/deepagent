## Install

<PackageTabs>
{`
\`\`\`bash npm
npm install @deepagent/wolfram-alpha
\`\`\`

\`\`\`bash yarn
yarn add @deepagent/wolfram-alpha
\`\`\`

\`\`\`bash pnpm

\`\`\`
`}

## Usage

```ts
import { WolframAlphaClient } from '@deepagent/wolfram-alpha'

const wolframAlpha = new WolframAlphaClient()
const res = await wolframAlpha.search('latest news')
```
