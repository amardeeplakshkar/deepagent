## Install

<PackageTabs>
{`
\`\`\`bash npm
npm install @deepagent/tavily
\`\`\`

\`\`\`bash yarn
yarn add @deepagent/tavily
\`\`\`

\`\`\`bash pnpm
pnpm add @deepagent/tavily
\`\`\`
`}
</PackageTabs>

## Usage

```ts
import { TavilyClient } from '@deepagent/tavily'

const tavily = new TavilyClient()
const res = await tavily.search('latest news')
```
