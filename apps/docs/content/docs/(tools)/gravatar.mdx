## Install

<PackageTabs>
{`
\`\`\`bash npm
npm install @deepagent/gravatar
\`\`\`

\`\`\`bash yarn
yarn add @deepagent/gravatar
\`\`\`

\`\`\`bash pnpm
pnpm add @deepagent/gravatar
\`\`\`

</PackageTabs>

## Usage

```ts
import { GravatarClient } from '@deepagent/gravatar'

const gravatar = new GravatarClient()
const profile = await gravatar.getProfileByIdentifier('my-email@example.com')
```
