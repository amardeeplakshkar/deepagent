## Install

<PackageTabs>
{`
\`\`\`bash npm
npm install @deepagent/apollo
\`\`\`

\`\`\`bash yarn
yarn add @deepagent/apollo
\`\`\`

\`\`\`bash pnpm

\`\`\`

## Usage

```ts
import { ApolloClient } from '@deepagent/apollo'

const apollo = new ApolloClient()
const res = await apollo.enrichPerson({
  linkedin_url: 'https://linkedin.com/in/fisch2'
})
```
