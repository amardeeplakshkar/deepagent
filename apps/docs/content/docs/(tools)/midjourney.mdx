- package: `@deepagent/midjourney`

- [imagine api docs](https://www.imagineapi.dev)

## Install

<PackageTabs>
{`
\`\`\`bash npm
npm install @deepagent/midjourney
\`\`\`

\`\`\`bash yarn
yarn add @deepagent/midjourney
\`\`\`

\`\`\`bash pnpm
pnpm add @deepagent/midjourney
\`\`\`
`}
</PackageTabs>

## Usage

```ts
import { MidjourneyClient } from '@deepagent/midjourney'

const midjourney = new MidjourneyClient()
const res = await midjourney.imagine({
  prompt: 'beautiful sunset over the ocean, oil painting, monet'
})
```
