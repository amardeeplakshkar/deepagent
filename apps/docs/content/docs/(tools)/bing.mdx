## Install

<PackageTabs>
{`
\`\`\`bash npm
npm install @deepagent/bing
\`\`\`

\`\`\`bash yarn
yarn add @deepagent/bing
\`\`\`

\`\`\`bash pnpm
pnpm add @deepagent/bing
\`\`\`
`}
</PackageTabs>

## Usage

```ts
import { BingClient } from '@deepagent/bing'

const bing = new BingClient()
const res = await bing.search('latest news')
```
