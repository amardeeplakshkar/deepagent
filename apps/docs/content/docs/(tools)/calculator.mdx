## Install

<PackageTabs>
{`
\`\`\`bash npm

\`\`\`

\`\`\`bash yarn
yarn add @deepagent/calculator

## Usage

```ts
import { calculator } from '@deepagent/calculator'

const res = await calculator({ expr: '1 + 1' })
```
